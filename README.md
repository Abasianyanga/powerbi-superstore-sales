## Superstore Sales Dashboard (Power BI)

This project analyzes sales performance using the Superstore dataset.

### Tools Used
- Excel (Data Cleaning)
- Power BI (Visualization)

### Data Preparation
- Cleaned raw sales data in Excel
- Fixed date formats
- Created delivery days column
- Standardized categories

### Key Insights
- Sales increased steadily over the years
- The West region generated the highest sales
- Consumer segment contributed the most revenue
- Average delivery time is approximately 4 days
